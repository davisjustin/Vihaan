Anagram

Start where you are, use what you have, do what you can.

The game pieces consist of a set of tiles with letters on them. Tiles are turned over one by one and players form words by combining unused tiles with existing words, thier own or others. The game has never been standardised and there exists a great many varieties of sets and rules. Anagram is now often played with tiles from another word game such as scrabble. Web-based version of this game is also available.

Rules

  • The tablets must be so placed in the box that the letters cannot be seen until drawn out.
  • To commence the game each player draws a letter from the box.
  • Then the player should find out the word within the limited time.
  • If he/she fails to find, then the next letter should be picked up.
  • Finally the word formed by the letters should be guessed out.
